# Logo Usage (Partner-Safe)

ARKTIK marks are proprietary and protected. This document provides high-level guidance only.

## Allowed (with permission or as directed by ARKTIK)
- Press usage in coverage about ARKTIK
- Partner usage under written agreement
- Event usage when ARKTIK is an official participant

## Not allowed
- Using ARKTIK logos in your product branding, social handles, or domains
- Implying endorsement, certification, or partnership
- Altering the logo, changing colors, or creating “inspired by” variations

If you need logo files or explicit approval, request permission via arktikeducation.org.
